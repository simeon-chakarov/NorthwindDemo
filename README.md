# Northwind Demo

ASP.NET Core solution containing:

- Web API
- MVC client
- Contracts project
- Unit tests
- CORS configuration
- Cancellation token support

## Run

1. Ensure SQL Server is running
2. Set both API and MVC as startup projects
3. Run solution

Default URLs:
- MVC: https://localhost:7265
- API: https://localhost:7013
